 - next we should figure out how we want our elements to display in different devices. for this example lets do small, medium, 
    and large size screens

    [drawing of designs for small, medium, large]

 - We can take our laytout designs and create a grid that encapsulates the different parts.

    [drawings with grids shown]

 - Css-Grid allows us to easily define each section of the grid using 'gridareas' and then associate our elements with one of these
    gridareas.

    Define the grid areas and how they are positioned in CSS:

        /* Small Screens */

        .grid-container {
            display: grid;
            grid-template-rows: 50px 50px 1fr 30px;
            grid-template-areas: 
                "header" 
                "sidebar" 
                "content" 
                "footer"
        }

        /* Medium Screens */

        @media only screen and (min-width: 768px) {
            .grid-container {
                grid-template-columns: 200px 1fr;
                grid-template-rows: 50px 1fr 30px;
                grid-template-areas: 
                    "header   header" 
                    "sidebar  content" 
                    "footer   footer";
            }
        }

        /* Large Screens */

        @media only screen and (min-width: 1200px) {
            .grid-container {
                grid-template-columns: 200px 1fr 1fr;
                grid-template-rows: 50px 1fr 30px;
                grid-template-areas: 
                    "header  header  header" 
                    "sidebar content sidebar2" 
                    "footer  footer  footer";
            }
        }
